# 🚀 DevBattle

A modern React-based web application that enables users to explore GitHub profiles, analyze developer activity, and compare two developers in a competitive “battle mode” based on real-time data.

---

## 🌟 Overview

DevBattle provides a seamless interface to fetch and visualize GitHub user data. It allows users to search for any developer, view their profile insights, explore repositories, and compare two developers based on their overall impact (measured through repository stars).

The application emphasizes clean UI, real-time API interaction, and an engaging comparison experience.

---

## ⚙️ Features

* 🔍 **Profile Search**
  Search any GitHub user and view their public profile details.

* 📊 **Repository Insights**
  Displays the latest repositories along with key metrics like stars and creation date.

* ⚔️ **Battle Mode**
  Compare two GitHub users side-by-side and determine a winner based on total stars.

* 🎨 **Dynamic Theming**
  Toggle between light and dark modes with adaptive background visuals.

* ⏳ **Loading & Error Handling**
  Smooth loading states and graceful handling of invalid users or API failures.

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Styling:** CSS (Custom + Theme-based variables)
* **Build Tool:** Vite
* **API:** GitHub REST API
* **Deployment:** Vercel

## 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/devbattle.git
cd devbattle
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```
