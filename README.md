# GitHub Battle App ⚔️

A React-based web application that allows users to search GitHub profiles, view repositories, and compare two users based on their GitHub activity.

## 🚀 Features

### 🔹 Level 1
- Search GitHub user
- Display profile (avatar, bio, join date, portfolio)
- Loading & error handling

### 🔹 Level 2
- Fetch repositories
- Show top 5 latest repos
- Human-readable date formatting
- Clickable repo links

### 🔹 Level 3 (Battle Mode)
- Compare two users
- Calculate total stars
- Highlight winner and loser

### 🎨 Custom Features
- Light/Dark theme toggle
- Dynamic background image based on theme
